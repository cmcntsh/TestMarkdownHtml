You can edit web pages using an online text editor in GitHub by navigating to the file in GitHub and pressing the period (.) key. The page will open up in an online text editor.

Once you have an index.html page in the main directory, you can turn on GitHub Pages to render the page directly from GitHub.

To turn on GitHub Pages, complete the following steps.

  • From the main page of the repository in GitHub go to Settings.
  • Go to the Pages section. (Found in the list along the left-hand side of the main Settings page.
  • Under the Branch section, click on the dropdown that says None and change it to Main.
  • Click Save.
